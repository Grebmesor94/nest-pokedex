# Ejecutar en desarrollo

1. Clonar repositorio
2. Ejecutar

```
npm install
```

3. Instalar nest CLI en caso de no tenerlo

```
npm i -g @nest/cli
```

4. Levantar base de datos

```
  docker compose up -d
```
